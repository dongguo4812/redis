安全Bug按照官网提示，升级成为6.0.8及以上。

> 6.0.7 被称作最后一个 6.× 版本，不过Redis 突然发布了紧急版本 6.0.8 ，Redis团队表示6.0.8版本升级迫切性等级为高:任何将Redis 6.0.7 与Sentinel或CONFIG REWRITE命令配合使用的人都会受到影响，应尽快升级。
